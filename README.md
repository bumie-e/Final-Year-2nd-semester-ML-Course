# Final-Year-1st-semester-ML-Course
This is the reference platform for code walkthrough

## Course Description

This course provides a practical, industry-oriented introduction to machine learning, focusing on how modern ML systems are designed, trained, evaluated, deployed, and maintained in real-world settings. Unlike purely theoretical or deep learning focused courses, this course emphasises classical and applied machine learning, decision-making, and end-to-end workflows used in industry.

Students will work with real datasets, industry-style tooling, and open-ended problems that mirror professional ML practice. The course prioritises thinking like an ML engineer or applied scientist, not just implementing algorithms.

Note: Advanced AI and Deep Learning topics (e.g. transformers, large neural networks) are intentionally de-emphasised, as they are covered in separate courses.

### Course Objectives
By the end of the course, students should be able to:
- Frame real-world problems as machine learning tasks
- Select appropriate models and evaluation strategies
- Understand bias-variance trade-offs and generalisation
- Build reproducible ML pipelines
- Diagnose model failures and data issues
- Communicate results to technical and non-technical stakeholders
- Understand ethical, business, and deployment constraints of ML systems

### Prerequisites
- Python programming
- Basic probability and statistics
- Linear algebra fundamentals

### Teaching Format

- **Lectures**: Conceptual framing + intuition
- **Labs**: Hands-on notebooks (JupyterLab, VSCode, GoogleColab)
- **Case studies**: Real industry scenarios
- **In-class discussions**: Trade-offs, failures, ethics

Students are expected to read independently; class time is used to go deeper, challenge assumptions, and apply concepts.

### Recommended Textbooks & Resources 
*While the course does not strictly follow one textbook, the following are highly recommended for theoretical intuition and mathematical grounding:*
- Pattern Recognition and Machine Learning by Christopher M. Bishop.
- Machine Learning: A Probabilistic Perspective by Kevin P. Murphy.
- The Elements of Statistical Learning by Trevor Hastie, Robert Tibshirani, and Jerome Friedman.
- Machine Learning by Tom Mitchell.
